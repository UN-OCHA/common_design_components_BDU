# Typography

Typography plays an important role in infographics and communications products. OCHA uses open-source Google Fonts for compatibility and to avoid licensing issues.

> **Roboto** and **Crimson Pro** are OCHA corporate fonts (for Latin alphabets).

---

## Primary typeface: Roboto

Roboto is the primary font family, providing various weights each with regular and italic versions. It comes in four sub-families:

<div style={{ display: 'flex', flexWrap: 'wrap', gap: '1rem' }}>
  <FontCard name="Roboto" family="Roboto, sans-serif" role="Primary — body text, UI, general use" />
  <FontCard name="Roboto Slab" family="Roboto Slab, serif" role="Headings" />
  <FontCard name="Roboto Condensed" family="Roboto Condensed, sans-serif" role="Infographics (save space)" />
  <FontCard name="Roboto Mono" family="Roboto Mono, monospace" role="Small text, quote cards, photo credits" />
</div>

> **Do not** use Roboto Slab, Roboto Condensed, or Roboto Mono as body text.

---

## Secondary typeface: Crimson Pro

<FontCard name="Crimson Pro" family="Crimson Pro, Georgia, serif" role="Print long-form body text" note="Use Crimson Pro, NOT Crimson Text" />

Crimson Pro is an open-source serif font used to complement Roboto, especially in print production and long body text for readability.

---

## Common users — no install needed

**Arial** is the font of choice for general use, as it ships with all major operating systems. Use Arial for any collaborative product needing frequent updates by multiple people (Word documents, spreadsheets, shared reports).

> **Do not** use the Arial Narrow font family.

---

## Type scale

The system uses a 9-step scale. All values are in `rem` relative to a 16px root.

## Do's and Don'ts

### Do not use these fonts

| Font | Why |
|---|---|
| **Arial Narrow** | Not included by default with the OS. Licensed with MS Office. |
| **Avenir** | Was the primary font before Roboto. Not default, licensed. |
| **Minion Pro** | Was the secondary font before Crimson Pro. Not default, licensed. |
| **Crimson Text** | Replaced by Crimson Pro which has more styles available. |

## CSS tokens reference

```css
/* Primary */
--cd-font--roboto: "Roboto", helvetica, arial, sans-serif;
--cd-font--roboto-slab: "Roboto Slab", helvetica, arial, sans-serif;
--cd-font--roboto-condensed: "Roboto Condensed", helvetica, arial, sans-serif;
--cd-font--roboto-mono: "Roboto Mono", ui-monospace, "SF Mono", Consolas, monospace;

/* Secondary */
--cd-font--crimson-pro: "Crimson Pro", Georgia, "Times New Roman", serif;

/* Multilingual */
--cd-font--noto-sans: "Noto Sans", sans-serif;
--cd-font--noto-kufi-arabic: "Noto Kufi Arabic", sans-serif;
--cd-font--noto-sans-sc: "Noto Sans SC", sans-serif;
--cd-font--dubai: "Dubai", "Noto Sans Arabic", sans-serif;
--cd-font--almarai: "Almarai", "Noto Sans Arabic", sans-serif;

/* Fallback */
--cd-font: helvetica, arial, sans-serif;
```
