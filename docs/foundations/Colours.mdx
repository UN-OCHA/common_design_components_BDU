# Colours

OCHA adopted the **UN Blue** (Pantone 2925) as its primary colour. The full palette is the updated United Nations colour palette.

> **A six-value ramp inspired by the UN blue will be sufficient for most design and infographic needs when combined with grey and black. Resist the temptation to add more colours.**
>
> — OCHA Branding Guidelines

## Primary brand colours

<div style={{ display: 'flex', gap: '2rem', flexWrap: 'wrap' }}>
  <Swatch name="UN Blue" hex="#009EDB" />
  <Swatch name="Black" hex="#000000" />
  <Swatch name="White" hex="#FFFFFF" />

The primary colour should be used predominantly in all OCHA visuals. Blue represents peace and evokes the clear, boundless sky shared by every person in every country.

### Accessible text variants

When using blue as text on a white background, the primary `#009EDB` does not meet WCAG AA contrast requirements. Use these darker variants instead:

<div style={{ display: 'flex', gap: '2rem', flexWrap: 'wrap' }}>
  <Swatch name="Blue AA" hex="#0077B8" aa />
  <Swatch name="Blue AAA" hex="#005392" aa />
</div>

| Context | Use this colour | Token |
|---|---|---|
| Large fills, hero backgrounds, brand moments | `#009EDB` | `--brand-primary` |
| **Text, links, buttons on white** | **`#0077B8`** | **`--brand-primary--text`** |
| Critical text requiring maximum contrast | `#005392` | `--brand-primary--text-strong` |

## Semantic colour tokens

Components use these `--brand-*` tokens, not raw hex values. This ensures accessibility and enables retheming.

| Token | Hex | Use |
|---|---|---|
| `--brand-primary` | `#009EDB` | Fills, backgrounds, brand moments |
| `--brand-primary--text` | `#0077B8` | Text on white (AA-safe) |
| `--brand-primary--text-strong` | `#005392` | Critical text (AAA-safe) |
| `--brand-primary--dark` | `#004987` | Hover / active states |
| `--brand-primary--light` | `#64BDEA` | Light fills |
| `--brand-highlight` | `#ED1847` | Accent / CTA |
| `--brand-highlight--text` | `#EB0045` | Accent text (AA-safe) |
| `--brand-success` | `#27833A` | Success states |
| `--brand-warning` | `#CF3F0B` | Warning states |
| `--brand-error` | `#EB0045` | Error states |
| `--brand-info` | `#0077B8` | Info states |
| `--brand-grey` | `#F2F2F2` | Neutral surfaces |
| `--brand-grey--text` | `#737373` | Muted text |
| `--brand-default-text-color` | `#4D4D4D` | Body text |
